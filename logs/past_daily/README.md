# Past Daily Logs

このディレクトリには、
過去の日記、古い日次ログ、後から再投入した履歴日記を置く。

`logs/past_daily/` は、
現在の `daily` とは別の履歴専用層である。
ここにある記述は高価値な入力資源だが、
現在の `current_state` や昇格回路には自動接続しない。

## 役割

- 過去日記を元の日付つきで保持する
- 過去の強い attractor や自己像を、現在状態と分離したまま参照できるようにする
- `meta/update/reingestion.md` に従って再読解するための正本置き場にする

## current daily との違い

- `logs/daily/`:
  現在運用中の日次状態と微細調整
- `logs/past_daily/`:
  過去の日記や、履歴入力として再投入した古い日次記録

## 原則

- ファイル名は原則 `YYYY-MM-DD.md` とし、同日に複数ある場合だけ連番を付ける
- 強く書かれていても、現在の真実や current_state の正本として扱わない
- 現在ロジックに接続したい場合は、必ず `meta/update/reingestion.md` を通す
- タイトルや要約はファイル名ではなく本文側に保持する
